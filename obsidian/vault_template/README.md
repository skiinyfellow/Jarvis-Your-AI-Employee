# Jarvis Obsidian Vault Template

This directory contains the Obsidian vault template for integrating
with Jarvis for knowledge management and note synchronization.

## Directory Structure

- **Daily Notes** - Daily summaries and logs
- **Tasks** - Task management and tracking
- **Knowledge** - Knowledge base and documentation
- **Projects** - Project tracking and status
- **References** - External references and links
