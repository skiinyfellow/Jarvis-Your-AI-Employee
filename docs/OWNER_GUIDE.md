# Owner Guide

Welcome to Jarvis! This guide will help you understand and manage your AI employee.

## Getting Started

### 1. Initial Setup

- Follow the installation guide
- Configure your Telegram bot token
- Set up your Obsidian vault
- Configure your Ollama models

### 2. Accessing Jarvis

- **Dashboard:** http://localhost:8000
- **Telegram:** Send `/start` to your Jarvis bot
- **Obsidian:** Sync with your configured vault

### 3. Adding Tasks

Use the dashboard or Telegram bot to:
- Create tasks
- Monitor progress
- View history

## Managing Users

Edit `.env` to adjust:

```
MAX_USERS=1
LICENSE_KEY=your_key
```

## Monitoring

Check the dashboard for:
- System status
- Active users
- Completed tasks
- System logs
